# SuperMenu
Powerful launcher for Windows (AutoHotkey v2)
⚡ SuperMenu v4.1 for AutoHotkey v2

SuperMenu — это мощный многофункциональный лаунчер и панель инструментов для Windows, написанная на AutoHotkey v2. Он объединяет в себе менеджер буфера обмена, быстрый доступ к системным утилитам, управление заметками и кастомизируемое меню запуска.

✨ Основные возможности

📋 История буфера обмена: Хранит до 20 последних текстовых записей.

⭐ Кастомное меню: Добавляйте свои файлы, папки или ссылки на сайты через удобный интерфейс.

🎨 Темы оформления: 5 стильных цветовых схем (Dark, Blue, Green, Red, Light).

📝 Быстрые заметки: Встроенный текстовый редактор с возможностью сохранения в выбранную папку.

⏱️ Таймер и Будильник: Простые инструменты для контроля времени с уведомлениями.

⚙️ Системные инструменты: Быстрый доступ к CMD/PowerShell (в т.ч. от админа), управление скрытыми файлами и принудительное завершение зависших окон.

🌐 Двуязычный интерфейс: Полная поддержка русского и английского языков.

⌨️ Горячие клавиши (Hotkeys)

Alt + Z — Открыть главное меню

Win + T — Запустить терминал

Win + Shift + A — Закрепить окно поверх всех

Win + Shift + K — "Убить" активный процесс

Win + Shift + H — Показать/скрыть скрытые файлы

Win + Shift + V — История буфера обмена

Win + Shift + J — Быстрые заметки

🚀 Установка

Установите AutoHotkey v2.

Скачайте файл SuperMenu.ahk.

Запустите скрипт двойным кликом.

⚡ SuperMenu v4.1 for AutoHotkey v2 (English)

SuperMenu is a powerful multi-tool launcher and productivity dashboard for Windows, built with AutoHotkey v2. It combines a clipboard manager, system utilities, note-taking, and a customizable app launcher into one compact script.

✨ Features

📋 Clipboard History: Tracks up to 20 recent text entries.

⭐ Custom Apps: Add your own files, folders, or URLs via a simple GUI.

🎨 Themes: 5 built-in color schemes (Dark, Blue, Green, Red, Light).

📝 Scratchpad: Quick notes editor with "Save As" functionality to custom paths.

⏱️ Timer & Alarm: Lightweight time management tools with visual alerts.

⚙️ System Tools: Quick CMD/PowerShell access (Admin supported), toggle hidden files, and "Force Kill" active windows.

🌐 Multi-language: Full Russian and English support.

🚀 How to use

Install AutoHotkey v2.

Download SuperMenu.ahk.

Run the script.
